Цель: понять, что такое Vue и как быстро начать.

Практическое задание: запустить приложение с Vite и изменить стартовый экран.

Что внутри:
- Что такое Vue простыми словами (реактивный UI).
- Когда Vue подходит лучше всего.

- Структура проекта: main.js, App.vue, components.
- DevTools для Vue.

Закрепление:
- Мини-практика: создать «Привет, Vue».
- Контрольные вопросы: зачем Vite, где точка входа.
- Типичные ошибки: путать HTML и шаблон Vue, запускать не ту команду.
