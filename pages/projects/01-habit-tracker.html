Проект A. Трекер привычек

Первый проект для закрепления Vue-базы: список привычек, ежедневные отметки и простой прогресс выполнения.

Цель проекта

Собрать рабочий мини-продукт на Vue 3 и повторить ключевые навыки: реактивные данные, события, формы, computed-свойства и сохранение состояния.

Функциональные требования

  • Добавление новой привычки (название + категория).
  • Отметка выполнения привычки за сегодня.
  • Удаление привычки.
  • Фильтрация: все / выполненные / невыполненные.
  • Отображение прогресса за день в процентах.

План реализации (по шагам)

  1. Создай массив привычек в `ref` и выведи список через `v-for`.
  2. Добавь форму создания привычки и валидацию пустого названия.
  3. Реализуй переключатель статуса выполнения через чекбокс.
  4. Сделай `computed` для процента выполненных привычек.
  5. Добавь фильтры и кнопку удаления.
  6. Сохрани состояние в `localStorage` (через `watch`).

Критерии готовности

  • Можно добавить, отметить и удалить привычку без перезагрузки.
  • После обновления страницы данные не теряются.
  • Прогресс обновляется корректно при каждом действии.
  • Интерфейс читаемый и аккуратный на desktop и mobile.

Техническое задание

  • Модель привычки: { id, title, category, doneToday, streak, createdAt }.
  • Категории: здоровье, обучение, работа, личное.
  • Ввод новой привычки блокируется, если название короче 3 символов.
  • Рассчитывай прогресс: completed / total * 100 через computed.
  • Сохраняй данные в localStorage ключом habit-tracker:v1.

Чеклист приемки

  1. Добавлены минимум 3 привычки разных категорий.
  2. После перезагрузки состояния и прогресс совпадают с предыдущим шагом.
  3. Фильтры «все/выполненные/невыполненные» не ломают счетчик прогресса.
  4. Удаление привычки обновляет список и процент без визуальных артефактов.
