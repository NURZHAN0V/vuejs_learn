Проект C. Каталог товаров

Построй каталог: список карточек, фильтры по категориям, поиск и сортировка по цене/рейтингу.

Обязательные функции

  • Загрузка mock-данных (локальный JSON/массив).
  • Поиск по названию товара.
  • Фильтр по категории и диапазону цены.
  • Сортировка: по возрастанию/убыванию цены и рейтингу.
  • Карточка товара с кнопкой «В избранное».

Что потренируешь

  • Комбинацию фильтров через несколько `computed` цепочкой.
  • Локальный стейт корзины/избранного через `reactive`.
  • Повторное использование компонентов (`ProductCard`, `FiltersPanel`).

Критерий завершения

Пользователь находит нужный товар за 2-3 действия: поиск + фильтр + сортировка работают совместно.

Структура данных

  • Товар: { id, title, category, price, rating, inStock, image }.
  • Минимум 20 товаров в 4 категориях для реалистичной фильтрации.
  • Поддержка пустых результатов поиска с понятным сообщением.
  • Состояние фильтров синхронизируется с query-параметрами URL.

Критерии качества

  1. Поиск и фильтры применяются вместе, не перетирая состояние друг друга.
  2. Сортировка стабильна и предсказуема при равных значениях цены/рейтинга.
  3. Карточки не «скачут» по высоте, сетка остается ровной.
  4. Кнопка избранного сохраняет состояние после перезагрузки страницы.
