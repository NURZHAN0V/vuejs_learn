Раздел 3. События и пользовательские действия

Учимся реагировать на клики, ввод, клавиши и отправку форм через v-on и методы компонента.

Цель урока

Освоить ключевые идеи раздела и закрепить их через короткую пошаговую практику.

Теория простыми словами

1) v-on / @

v-on подписывает элемент на событие. Сокращенная запись — @. Пример: @click, @input, @submit.

2) Методы компонента

Логику действий лучше выносить в методы: код чище, проще тестировать и переиспользовать.

3) Модификаторы событий

.stop, .prevent, .once, .self позволяют управлять поведением события прямо в шаблоне.

4) Модификаторы клавиш и мыши

Можно ловить конкретные клавиши и кнопки мыши: @keyup.enter, @click.right и другие.

5) Передача аргументов в обработчик

В обработчик можно передавать данные и событие: @click="add(5, $event)".

Практика по шагам

Практическое задание: сделать счетчик с горячими клавишами и кнопкой сброса.

Мостик: шаги идут от простого клика к полноценному сценарию ввода и клавиш, чтобы ты видел, как одно действие логично наращивается другим.

Шаг 1. Простой клик по кнопке

<template>
  <button @click="increment">+1</button>
  <p>Счет: {{ count }}</p>
</template>

<script setup>
import { ref } from 'vue'
const count = ref(0)
const increment = () => {
  count.value++
}
</script>

Шаг 2. Добавь кнопку сброса и аргументы

<button @click="changeBy(5)">+5</button>
<button @click="reset">Сброс</button>

<script setup>
const changeBy = (step) => {
  count.value += step
}
const reset = () => {
  count.value = 0
}
</script>

Шаг 3. Горячие клавиши

<input
  placeholder="Нажми Enter для +1, Esc для сброса"
  @keyup.enter="increment"
  @keyup.esc="reset"
/>

Шаг 4. Модификаторы событий

<form @submit.prevent="save">
  <button type="submit">Сохранить</button>
</form>

<div @click="outer">
  <button @click.stop="inner">Не всплывает наверх</button>
</div>

<button @click.once="showOnce">Сработает один раз</button>
<div @click.self="onBackdrop">...</div>

Шаг 5. Полный пример счетчика

<template>
  <div class="space-y-2">
    <p>Счет: {{ count }}</p>
    <button @click="increment">+1</button>
    <button @click="changeBy(5)">+5</button>
    <button @click="reset">Сброс</button>
    <input @keyup.enter="increment" @keyup.esc="reset" />
  </div>
</template>

<script setup>
import { ref } from 'vue'
const count = ref(0)
const increment = () => count.value++
const changeBy = (step) => (count.value += step)
const reset = () => (count.value = 0)
</script>

Практическое задание

Сделай счетчик с горячими клавишами и кнопкой сброса.

  1. Кнопка +1 увеличивает значение на 1.
  2. Кнопка +5 увеличивает значение на 5 через аргумент функции.
  3. Кнопка Сброс ставит значение в 0.
  4. Enter увеличивает счет, Esc сбрасывает.
  5. Добавь форму с @submit.prevent, чтобы не было перезагрузки страницы.

Типичные ошибки

  • Вызывать метод сразу в шаблоне вместо передачи обработчика.
  • Забывать .prevent у формы и получать перезагрузку страницы.
  • Использовать слишком много модификаторов без понимания порядка обработки событий.
